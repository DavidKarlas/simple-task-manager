Welcome to the SimpleTaskManager for OpenStreetMap

Changes in 0.10.0:

New features:
  • Changed layout for the task-details view
  • Highlight assigned tasks
  • Uploading geometries now supports multi-polygons
Bug-Fixes:
  • New projects with 0 max. process points are no longer possible
  • Tasks with near-zero coordinates do not appear anymore
  • Assigned users are now getting unassigned when leaving a project
Internals:
  • Use user-ID instead of user name
  • Removed API version v1, v2 and v2.1
  • Reset database before each server test method
  • Use journald as docker logging-driver
  • Use GeoJSON to store task geometries
