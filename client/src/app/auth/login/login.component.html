Welcome to the SimpleTaskManager for OpenStreetMap

Changes in 1.1.0-dev:

New features:
  • Translation of STM into Chinese, German and Japanese
  • Button to open task in iD
  • Button to directly set a task as 'done'
Bug fixes:
  • Missing feedback on overpass error
Minor changes:
  • Zoom to selected task when not visible on the screen
Internals:
  • Store project-ID in task database table
  • Create project and tasks within one HTTP request
  • Introduce the context to add a logging trace-ID
  • Better server side logging
  • Reconnect to database after connection failure
