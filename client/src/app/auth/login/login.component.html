Welcome to the SimpleTaskManager for OpenStreetMap

Changes in 1.0.0:

New features:
  • Live updates of changed projects and tasks
  • Edit name and description of a project
  • Load tasks from remote source (e.g. uploaded file or overpass-api)
  • Add info (white) and warning (yellow) notifications next to errors (red)
Bug-Fixes:
  • Minor UI and styling bugs
Internals:
  • API upgrade from v2.2 to v2.3
  • Proper unsubscription in webclient
  • Updated dependencies
  • Websocket connections for server->client communication
  • Print stack traces on errors
