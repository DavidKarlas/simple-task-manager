Welcome to the SimpleTaskManager for OpenStreetMap
New features in 0.8.0:
  • Upload GPX, OSM or GeoJSON files when creating a project
  • Only the owner of a project can see the invitation form
  • More and better input-field validation
  • Working alone on a project doesn't require assignment
  • You can add a project description when creating a project
Internals:
  • Wrapping of error within the server
  • Added documentation about client & server architecture
  • Use Array-Types in database instead of large strings
  • Prevent SQL injection
